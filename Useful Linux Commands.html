Linux Commands :



1. pwd : When you first open the terminal, you are in the home directory of your user. To know which directory you are in, you can use the ‚Äúpwd‚Äù command. It gives us the absolute path, which means the path that starts from the root. The root is the base of the Linux file system. It is denoted by a forward slash( / ). The user directory is usually something like ‚Äú/home/username‚Äù.

2. ls ‚Äî Use the ‚Äúls‚Äù command to know what files are in the directory you are in. You can see all the hidden files by using the command ‚Äúls -a‚Äù. ‚Äúls -l‚Äù can be used to see long listing of file with the information .

3. cd ‚Äî Use the ‚Äúcd‚Äù command to go to a directory. For example, if you are in the home folder, and you want to go to the downloads folder, then you can type in ‚Äúcd Downloads‚Äù. Remember, this command is case sensitive, and you have to type in the name of the folder exactly as it is. To go back from a folder to the folder before that, you can type ‚Äúcd ..‚Äù . The two dots represent back.

4. mkdir & rmdir ‚Äî Use the mkdir command when you need to create a folder or a directory. For example, if you want to make a directory called ‚ÄúDIY‚Äù, then you can type ‚Äúmkdir DIY‚Äù. If you want to create a directory named ‚ÄúDIY Hacking‚Äù, then you can type ‚Äúmkdir DIY\ Hacking‚Äù. Use rmdir to delete a directory. But rmdir can only be used to delete an empty directory. To delete a directory containing files, use rm.

5. rm ‚Äî Use the rm command to delete files and directories. Use ‚Äúrm -r‚Äù to delete just the directory. It deletes both the folder and the files it contains when using only the rm command.

6. rf ‚Äî deletes directory forcefully. It means a file or directory will be deleted anyhow even if it has read-only permission. To delete a file forcefully, use command: rm -f <file name>

7. touch ‚Äî The touch command is used to create a file. It can be anything, from an empty txt file to an empty zip file. For example, ‚Äútouch new.txt‚Äù.

8. man & ‚Äî help ‚Äî To know more about a command and how to use it, use the man command. It shows the manual pages of the command. For example, ‚Äúman cd‚Äù shows the manual pages of the cd command. Typing in the command name and the argument helps it show which ways the command can be used (e.g., cd ‚Äìhelp).

9. cp ‚Äî Use the cp command to copy files through the command line. It takes two arguments: The first is the location of the file to be copied, the second is where to copy.

10. mv ‚Äî Use the mv command to move files through the command line. We can also use the mv command to rename a file. For example, if we want to rename the file ‚Äútext‚Äù to ‚Äúnew‚Äù, we can use ‚Äúmv text new‚Äù. It takes the two arguments, just like the cp command.

11. locate ‚Äî The locate command is used to locate a file in a Linux system, just like the search command in Windows. This command is useful when you don‚Äôt know where a file is saved or the actual name of the file. Using the -i argument with the command helps to ignore the case.

12. echo ‚Äî The ‚Äúecho‚Äù command helps us move some data, usually text into a file. For example, if you want to create a new text file or add to an already made text file, you just need to type in, ‚Äúecho hello, my name is alok >> new.txt‚Äù. You do not need to separate the spaces by using the backward slash here, because we put in two triangular brackets when we finish what we need to write.

13. cat ‚Äî Use the cat command to display the contents of a file. It is usually used to easily view programs. The cat command is a multi-purpose utility in the Linux system. It can be used to create a file, display content of the file, copy the content of one file to another file, and more. The cat command is also used as a filter. To filter a file, it is used inside pipes. eg : cat <fileName> | cat or tac | cat or tac |. . .

14. sudo ‚Äî A widely used command in the Linux command line, sudo stands for ‚ÄúSuperUser Do‚Äù. So, if you want any command to be done with administrative or root privileges, you can use the sudo command. For example, if you want to edit a file like viz. alsa-base.conf, which needs root permissions, you can use the command ‚Äî sudo nano alsa-base.conf. You can enter the root command line using the command ‚Äúsudo bash‚Äù, then type in your user password. You can also use the command ‚Äúsu‚Äù to do this, but you need to set a root password before that. For that, you can use the command ‚Äúsudo passwd‚Äù(not misspelled, it is passwd). Then type in the new root password.

15. df ‚Äî Use the df command to see the available disk space in each of the partitions in your system. You can just type in df in the command line and you can see each mounted partition and their used/available space in % and in KBs. If you want it shown in megabytes, you can use the command ‚Äúdf -m‚Äù.

16. du ‚Äî Use du to know the disk usage of a file in your system. If you want to know the disk usage for a particular folder or file in Linux, you can type in the command df and the name of the folder or file. For example, if you want to know the disk space used by the documents folder in Linux, you can use the command ‚Äúdu Documents‚Äù. You can also use the command ‚Äúls -lah‚Äù to view the file sizes of all the files in a folder.

17. zip, unzip ‚Äî Use zip to compress files into a zip archive, and unzip to extract files from a zip archive.

18. gzip ‚Äî The gzip command is used to truncate the file size. It is a compressing tool. It replaces the original file by the compressed file having ‚Äò.gz‚Äô extension. gzip <file1> <file2> <file3>‚Ä¶

19. uname ‚Äî Use uname to show the information about the system your Linux distro is running. Using the command ‚Äúuname -a‚Äù prints most of the information about the system. This prints the kernel release date, version, processor type, etc.

20. tar ‚Äî Use tar to work with tarballs (or files compressed in a tarball archive) in the Linux command line. It has a long list of uses. It can be used to compress and uncompress different types of tar archives like .tar, .tar.gz, .tar.bz2,etc. It works on the basis of the arguments given to it. For example, ‚Äútar -cvf‚Äù for creating a .tar archive, -xvf to untar a tar archive, -tvf to list the contents of the archive, etc.

21. apt-get ‚Äî Use apt to work with packages in the Linux command line. Use apt-get to install packages. This requires root privileges, so use the sudo command with it. For example, if you want to install the text editor jed (as I mentioned earlier), we can type in the command ‚Äúsudo apt-get install jed‚Äù. Similarly, any packages can be installed like this. It is good to update your repository each time you try to install a new package. You can do that by typing ‚Äúsudo apt-get update‚Äù. You can upgrade the system by typing ‚Äúsudo apt-get upgrade‚Äù. We can also upgrade the distro by typing ‚Äúsudo apt-get dist-upgrade‚Äù. The command ‚Äúapt-cache search‚Äù is used to search for a package. If you want to search for one, you can type in ‚Äúapt-cache search jed‚Äù(this doesn‚Äôt require root).

22. chmod ‚Äî Use chmod to make a file executable and to change the permissions granted to it in Linux. To make a file executable, you can use the command ‚Äúchmod +x numbers.py‚Äù in this case. You can use ‚Äúchmod 755 numbers.py‚Äù to give it root permissions or ‚Äúsudo chmod +x numbers.py‚Äù for root executable.

24. ping ‚Äî Use ping to check your connection to a server. Wikipedia says, ‚ÄúPing is a computer network administration software utility used to test the reachability of a host on an Internet Protocol (IP) network‚Äù. Simply, when you type in, for example, ‚Äúping google.com‚Äù, it checks if it can connect to the server and come back. It measures this round-trip time and gives you the details about it. The use of this command for simple users like us is to check your internet connection. If it pings the Google server (in this case), you can confirm that your internet connection is active!

25. ps/top ‚Äî The ps / top commands allow you to list the processes running on your computer.

26. kill ‚Äî l The kill command can be used to end a running task. You may want to end a program that‚Äôs either not responding or that‚Äôs running in the background. Either way, you‚Äôll have to find out the PID of the process (you can do that with ps or top).

27. service ‚Äî The service command can be used to manage services on your computer. It‚Äôs quite versatile in a way that you can start, stop, and restart a service as well as print details and a list of all available services on your computer.

28. grep ‚Äî A command, that allows you to search for specific characters in a string, is called grep. It will simply print all lines where the string you‚Äôve searched for appears. I have also made the page of grep commands in my profile.

29. head ‚Äî The head command is used to display the content of a file. It displays the first 10 lines of a file.

30. tail ‚Äî The tail command is similar to the head command. The difference between both commands is that it displays the last ten lines of the file content. It is useful for reading the error message. eg : tail txt.log OR tail -f txt.log .

31. more ‚Äî The more command is quite similar to the cat command, as it is used to display the file content in the same way that the cat command does. The only difference between both commands is that, in case of larger files, the more command displays screenful output at a time.

33. su ‚Äî The su command provides administrative access to another user. In other words, it allows access of the Linux shell to another user.

34. id ‚Äî The id command is used to display the user ID (UID) and group ID (GID).

35. passwd ‚Äî The passwd command is used to create and change the password for a user.
